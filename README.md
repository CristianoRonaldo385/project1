# Интернет-магазин

Система управления интернет-магазином с возможностью поиска товаров по категориям и фильтрации покупателей по сумме покупок.

## Функциональность

- Управление товарами, категориями и покупателями
- Поиск товаров по категории
- Фильтрация покупателей по сумме покупок

## Сущности

### Товар (Product)
- Название
- Цена
- Категория
- Количество на складе

### Категория (Category)
- Название категории
- Описание

### Покупатель (Customer)
- Имя
- Email
- Общая сумма покупок
